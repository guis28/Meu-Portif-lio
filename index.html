Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, para exibir uma coleção pessoal de livros favoritos. O objetivo é criar um ambiente virtual para compartilhar seus livros preferidos, com informações adicionais.

A estrutura do site é baseada em HTML, que define a semântica e layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, com cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro tem seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível, adaptando a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS criar um site acessível e esteticamente agradável sem precisar de scripts complexos, com foco na uso e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores ajuda para a atmosfera acolhedora do site, incentivando explorar a coleção de livros.

Minha Biblioteca: Uma Webpage Personalizada

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simulando uma narrativa centrada no tema da Inteligência Artificial. O objetivo é engajar os jogadores em um cenário fictício abordando o avanço da IA e suas consequências para a humanidade, destacando uma reflexão ética sobre a tecnologia.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, caixa de perguntas, alternativas de resposta, e o resultado final baseado nas escolhas do jogador. O CSS é usado para estilizar a página, empregando um esquema de cores futurista e layout responsivo, melhorarando a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando o DOM para atualizar o conteúdo da página conforme as escolhas do jogador, e determinando o fluxo da narrativa através de várias perguntas e respostas.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, detalhes do livro. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível, adaptando a diferentes tamanhos de tela.

Os scripts JavaScript incluem módulos para gerar nomes aleatórios, definir as perguntas e as lógicas de jogo, como iniciar o jogo, apresentar perguntas e alternativas, processar as escolhas dos jogadores, e exibir o resultado final. Este design modular facilita a manutenção e expansão futura do jogo. A importância das ferramentas usadas (HTML, CSS, JavaScript) reside em criar uma interação, que não apenas entretém, mas também educa sobre a complexidade ética e social da IA, incentivando a reflexão crítica sobre o papel da tecnologia em nossas vidas.

Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma aplicação educativa interativa focada em ensinar conceitos básicos de astronomia. Usa uma série de scripts ligados a diferentes atores (sprites) e cenários para criar um aprendizado envolvente. O objetivo é familiarizar os usuários com tópicos como as fases da Lua, eclipses, visibilidade de diferentes constelações dependendo da localização geográfica na Terra, e a forma da Terra, através de interações e visualizações dinâmicas.

Cada ator no projeto tem scripts que definem seu comportamento em resposta a tais eventos, como o início do programa (quando a bandeira é clicada), cliques do usuário, ou mensagens recebidas. Eles reagem com ações, como mudanças de cenário, movimentos de atores, exibição ou ocultação de atores, e a apresentação de diálogos informativos. Exemplo: ao clicar em determinados atores, os usuários podem "receber" uma prova, mudando o cenário e exibição de informações relevantes ou a visualização de uma tal constelação.

A importância das ferramentas usadas, como os blocos de código visual do Scratch, reside na sua acessibilidade e capacidade de engajar aprendizes de diferentes idades no pensamento computacional e explorando conceitos científicos de forma interativa. O uso de cenários, movimentos, e diálogos dinâmicos ajuda, criando um aprendizado mais rica e memorável, encorajando os usuários a explorar e interagir com o conteúdo de maneira ativa, em vez de passiva.
